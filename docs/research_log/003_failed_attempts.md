# 003 — Failed attempts and negative results

## Date
2026-06-19

## Failed attempt 1: raw one-day return
Raw one-day return was too noisy as a standalone signal. It can be included as an input but should not be presented as the core idea.

## Failed attempt 2: overstating PyTorch
A small PyTorch MLP looks more sophisticated on paper, but in this dataset it is not automatically more credible than ridge or random forest. The model is useful as a comparison point, not as proof of deep ML alpha.

## Failed attempt 3: agentic framing
The first version called the pipeline "agentic". That was inaccurate. The code executes a fixed workflow and does not make independent research decisions.

## Current action
Rename the framing to reproducible research pipeline and move human reasoning into research notes and report sections.
