# Research note template

## Hypothesis

Describe the economic or behavioral reason this signal might work.

## Dataset

Define the universe, time range, data source, missing-data policy, lagging rules, and point-in-time controls.

## Features

List feature definitions and how each one is lagged to avoid look-ahead bias.

## Validation

Include IC/rank-IC, decay, walk-forward splits, turnover/cost assumptions, and regime/subperiod checks.

## Results

Report both positive and negative results. Include failure cases.

## Next experiments

List what would be tested next and why.
